Welcome {{user.username}}!

GUESSence is a two player game where you describe cities in the world ONLY in English please. To turn off the push notifications please logout of the game.

Describer: describes a city ONLY using adjective/verb + common noun. Please do not use any NAMES including nationalities, languages, cardinal directions, locations, people or organizations.

Guesser: guesses a city ONLY using city names or "no clue" as response.

The describer starts the game and obtains a city name and a set of words (Taboo words) that may not be used in the game. The guesser submits a city names that s/he thinks is being described. For both roles please do not use numbers and punctuation, such as ":)".

If you run out of hints/guesses, you can start a new game by hitting "New Game"

Example Game: Lisbon, Portugal

Taboo words: fishing sardine earthquake football history oil museum olives port

Time: 2 minutes to submit hint/guess

Timeup: - 0.5 points on your score

Score: successful games + ( 0.1 x number of unsuccessful games)
